This package contains the AMT 1.2.0 with 

- pre-compiled MEX files for Windows (64 bit) and Linux (64 bit)
- following third-party toolboxes:
  - LTFAT 2.4.0-win64
  - SOFA API MO 1.1.1
  - SFS Matlab 2.4.0
  - Circular Statistics Toolbox
  - Binaural SH Toolbox
  
Tested on 
- Windows 10 (64 bit), Matlab 2016b, Curve fitting toolbox 3.5.3, Global optimization toolbox 3.4, Optimization toolbox 7.4, Signal processing toolbox 7.2, Statistics and ML toolbox 10.2 , Computer Vision toolbox
- Ubuntu 20.04(LTS), Matlab 9.2 (2017a), Curve fitting toolbox 3.5.5, Global optimization toolbox 3.4.2, Optimization toolbox 7.6, Signal processing toolbox 7.4, Statistics and ML toolbox 11.1, Computer Vision toolbox, Communication Systems toolbox
